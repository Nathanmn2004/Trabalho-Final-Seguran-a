Modo do servidor …

Vulnerável: sem proteção alguma — plaintext, sem bloqueios.
Protegido: aplica as proteções selecionadas abaixo.

Proteções ativas

Hash bcrypt
Senha armazenada com hash — cada verificação leva ~100ms, reduz de milhares para ~10 t/s
Bloqueio por IP
Bloqueia o IP após cada falha com exponential backoff (1s → 2s → 4s… até 5min)
Bloqueio por username
Bloqueia o usuário após cada falha — eficaz mesmo contra ataques distribuídos com IPs diferentes
